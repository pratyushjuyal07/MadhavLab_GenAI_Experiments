# MusicGen Indian Classical Experiments

## Objective
1. Explore generative music model architecture, training and conditioning using Meta MusicGen
2. Gain experience with fine tuning a model using a clean dataset
3. Develop intuition for relations between training data and generation quality

## Model
facebook/musicgen-small

## Experiments
- Prompt-based music generation
- Indian classical conditioning
- Hyperparameter experimentation

## Observations
- Specific prompts improved coherence
- Longer generations introduced noise
- Instrument consistency varied across prompts
